AIpy's GUI

AIpy's Graphical Users Interface (GUI) is build using NodeJS and Electron, meaning it is compatible with almost every OS like Linux and his Distros (Ubuntu, Debian, and many more...), Windows or MacOS. ElectronJS uses the technology of the web to create offline apps, making it easy to web developers to quickly create innovative apps using their knowledge of HTML development.
For building this project from source, you require ElectronJS and Electron-forge to be installed on your dev folder
