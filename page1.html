At the Beginning


How everything started

AIpy started as the idea for a school project to make a robotic arm capable to see and recognize objects using the Raspberry Pi and python. The purpose of this project was to make me learn about programming, setting up hardware and 3D modeling using Fusion 360.
During my researches, I discovered OpenCV, a python library used in image recognition. It is relatively complete and can be easily modded to use custom images. OpenCV is powerful and under BSD License, so he can be used on a variety of computer, in this case a Raspberry Pi.
I chose to use python as a primary programming language, because it's really simple to import external module, like Matplotlib, NumPy or OpenCV. It is also a widely supported language in the Tech world, like on phones and PCs, for example the Pi. Making it available to the Pi was one of my major concern, the other one was to make it as easy to build and use.
